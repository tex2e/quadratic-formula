二次方程式の解計算ツール

\(x^2 + \) \(x + \) \( = 0\)
\[{}\]

よくある質問

(1) どうやって使うのですか。

フォームに数字を入力して「計算」ボタンを押すか「Enterキー」を押すと、二次方程式の解を求めます。

例えば、\(a = 2, b = 3, c = -4\) を入力すると、 \( x = \frac{-3 \pm \sqrt{41}}{4} \) という解が得られます。

(2) 虚数解は計算できますか。

はい。虚数解も計算できます。

(2) 係数に少数を入れた場合でも計算できますか。

はい。誤差が含まれる可能性がありますが、正しく計算されます。

(3) \(a = 0\) や \(a = b = 0\) のときはどうなりますか。

\(a = 0\) のときは、一次方程式として計算します。 \(a = b = 0\) のときは、\(x = \mathrm{N/A}\)（Not Available）と表示します。

(4) 二次方程式の解って何ですか。

二次方程式には二つ解が存在します。 また、二つの解を求めるための公式（下の式）を二次方程式の解の公式と呼びます。 \[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]
